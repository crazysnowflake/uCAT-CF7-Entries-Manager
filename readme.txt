== Description ==

Save all submitted data from Contact Form 7 to database. All saved data displayed on admin as filterable table.

== Installation ==

Installing "uCAT - Contact Form 7 Entries Manager" can be done either by searching for "uCAT - Contact Form 7 Entries Manager" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
